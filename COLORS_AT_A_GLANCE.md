## Most Common Combinations

### Button (Primary CTA)
```
Background: #DAFF0D (Lime)
Text:       #1A1A1F (Dark BG)
Shadow:     rgba(218, 255, 13, 0.2) + rgba(255, 216, 90, 0.2)
Hover BG:   #E5FF4D (Light Lime)
```

### Button (Secondary)
```
Background: transparent
Border:     #B9A7FF (Soft Lavender)
Text:       #E5E4E8 (Light Gray)
Hover BG:   rgba(106, 75, 207, 0.1)
```

### Badge
```
Background: rgba(106, 75, 207, 0.1)
Border:     #B9A7FF (Soft Lavender)
Icon:       #DAFF0D (Lime)
Text:       #E5E4E8 (Light Gray)
```

### Card
```
Background: #F9F9F9 (Soft White)
Border:     [accent color] + 40% opacity
Text:       #1A1A1F (Dark BG)
Label:      #6B6780 (Medium Gray)
```

### Heading on Dark
